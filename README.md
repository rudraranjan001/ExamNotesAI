## 📖 About The Project

Preparing for exams can be stressful when managing scattered notes, multiple subjects, and limited time.

**ExamNotesAI** simplifies this process by offering:

- 🧠 AI-generated study notes
- 📂 Organized learning resources
- 🔐 Secure user authentication
- 📱 Fully responsive design
- ⚡ Fast and scalable deployment
- 🎯 Productivity-focused experience

This platform is built to make learning simpler, faster, and more effective.
